# PhytoScanAI

Application d'analyse de maladies des plantes par intelligence artificielle.

## Architecture

- **backend/** - API FastAPI (Python)
- **frontend/** - Interface utilisateur (React + Vite)
- **data/** - Datasets, knowledge base, vector store
- **notebooks/** - Expérimentations Data Science
- **evaluation/** - Tests de qualité IA (DeepEval)
- **infra/** - Monitoring & MLOps (Prometheus, Grafana, MLFlow)

## Lancement

```bash
docker-compose up --build
```
